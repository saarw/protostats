Helping app developers pick the right protocol by crowd-sourcing connectivity analytics from the world's mobile operators.

Download the Android app now to test your phone's and operator's connectivity. Completing the tests gives you access to the aggregated results!

ProtoStats was created as personal holiday hacking project by Swedish software engineer William Saar to help answer questions about mobile app development and play with new technologies.

The client is written in Android Java, the backend in Go, and this website uses AngularJS.
